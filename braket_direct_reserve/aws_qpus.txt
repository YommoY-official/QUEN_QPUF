AWS Braket QPU reference
========================
Drop-in values for DEVICE_NAME / DEVICE_ARN in submit_QPUF_ntarg.py.
Qubit counts are nominal — query device.properties.paradigm.qubitCount at
runtime for the authoritative number.

Notes
-----
- Region must match the ARN; the qiskit-braket-provider picks this up from
  the ARN, but boto3 / AwsDevice calls need the right AWS_REGION.
- MCM + reset: confirmed on IonQ Forte-1 via Braket Direct. Other devices
  may transpile reset to measure + conditional-X, or reject it — check
  before relying on register reuse.
- All-to-all connectivity: IonQ and AQT. Rigetti and IQM are fixed-lattice,
  so transpile will insert SWAPs.
